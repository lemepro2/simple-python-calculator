# Simple Python Calculator

A simple calculator that keeps running until you type `exit`.

## What It Does
- Add, subtract, multiply, divide
- Only works with **whole numbers** (like 5, 10, -3)
- Stops if you type `exit`
- Shows error if you divide by zero

## How to Use It (No Commands Needed!)

1. **Download the file**  
   → Click the green **Code** button → **Download ZIP**

2. **Unzip the folder** on your computer

3. **Open `main.py` with Python**  
   - If you have Python installed:  
     Double-click `main.py` **or** right-click → "Open with" → Python  
   - Or open your terminal/command prompt and type:
     ```bash
     python main.py
